# flavor_memo_app

이 프로젝트는 인프런 강의 **[[바이브코딩] Flutter 실무 레벨업: 3단계 빌드 변형(Flavor)과 코드 수정 없는 환경 전환](https://inf.run/oByuW)**의 스페셜 섹션에서 사용되는 소스 코드입니다.

## 시작하기

본 실습은 `starter` 브랜치에서 시작하는 것을 권장합니다. 아래의 명령어를 사용하여 프로젝트를 복제하고 `starter` 브랜치로 전환하세요.

### 방법 1: 복제와 동시에 브랜치 지정
```bash
git clone -b starter https://github.com/junsuk5/flutter_build_variants_course.git
```

### 방법 2: 복제 후 브랜치 전환
```bash
git clone https://github.com/junsuk5/flutter_build_variants_course.git
cd flutter_build_variants_course
git checkout starter
```
